# Лабораторная работа №6

**Тема:** "Генерация и обработка исключительных ситуаций"

## Задание

Продемонстрировать генерацию исключений, в которой будет осуществляться проверка ввода формата даты.
Например, необходимо написать дату в формате дд.мм.гг. Если  введен другой формат даты, то  сгенерировать исключительную ситуацию. 
В сообщение об ошибке должна входить информация о том, что введена  не указанная информация.
